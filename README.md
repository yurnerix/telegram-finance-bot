# 💰 Telegram Finance Bot

🤖 Telegram-бот для управления личными финансами: учет доходов, расходов, баланса и целей накопления. 

👉 Попробовать бота: **@finance_yurnerix_bot**

---

## 🚀 Возможности

- 📥 Добавление доходов  
- 📤 Добавление расходов  
- 📊 Просмотр баланса  
- 🎯 Установка целей накопления  
- 📈 Финансовая сводка  
- 👤 Регистрация пользователей  
- 🤖 Управление через Telegram команды

## 🤖 Команды бота

### 🔹 Основные

#### /start
Запуск бота и вывод приветственного сообщения

#### /help
Справка по всем доступным командaм

---

### 👤 Пользователь

#### /register Имя
Регистрация пользователя

Пример:
#### /register Юрий

---

### 💰 Доходы

#### /addincome Сумма Источник
Добавляет доход пользователя

Примеры:
#### /addincome 50000 зарплата
#### /addincome 12000 подработка

---

### 💸 Расходы

#### /addexpense Сумма Категория Описание
Добавляет расход пользователя

Примеры:

#### /addexpense 2500 продукты супермаркет
#### /addexpense 800 транспорт такси

---

### 📊 Баланс

#### /balance
Показывает:
- общую сумму доходов  
- общую сумму расходов  
- текущий баланс  

---

### 🎯 Цели

#### /setsavinggoal Сумма Название
Устанавливает цель накопления

Примеры:

#### /setsavinggoal 100000 отпуск
#### /setsavinggoal 80000 ноутбук

---

### 📈 Сводка

#### /summary
Показывает:
- сумму доходов  
- сумму расходов  
- текущий баланс  
- цель накопления  
- сколько осталось накопить  
- рекомендации по накоплению  

---

## ⚠️ Важные правила

1. Сначала выполните регистрацию
2. Суммы вводятся только числами  
3. В описаниях можно использовать слова через пробел  
4. При ошибке проверьте формат команды

## ▶️ Запуск проекта

### 1. Клонировать репозиторий
git clone https://github.com/yurnerix/telegram-finance-bot.git
### 2. Собрать проект
mvn clean install
### 3. Запустить
java -jar target/*.jar
