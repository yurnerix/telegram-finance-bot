# 💰 Telegram Finance Bot

🤖 Telegram-бот для управления личными финансами: учет доходов, расходов, баланса и целей накопления.

👉 Попробовать бота: **@finance_yurnerix_bot**

---

## 🚀 Возможности

- 📥 Добавление доходов  
- 📤 Добавление расходов  
- 📊 Просмотр баланса  
- 🎯 Установка целей накопления  
- 📈 Финансовая сводка  
- 👤 Регистрация пользователей  
- 🤖 Управление через Telegram команды  

---

## 🧠 Архитектура и подход

Проект реализован с упором на **чистую архитектуру и практики backend-разработки**:

- Принципы **SOLID**
- ООП (инкапсуляция, наследование, полиморфизм)
- Паттерны проектирования:
  - Command
  - Factory
- Разделение ответственности (handler / service / config / model)
- Масштабируемая структура
